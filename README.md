# Orçamento Pessoal v0.1

App de controle de orçamento pessoal em C++/Qt6 para Linux (Fedora/KDE Plasma).

## Requisitos

- Fedora 43 (ou superior)
- Qt6 (Widgets e Charts)
- Compilador GCC 15.2.1 ou superior

Instale as dependências:

```bash
sudo dnf install qt6-qtbase-devel qt6-qtcharts-devel gcc-c++ make -y
